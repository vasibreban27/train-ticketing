Train Ticketing - Postman Endpoint Tests
Base URL: http://localhost:9095

Note:
- Use the paths exactly as shown:
  /trains
  /bookings
  /routes
  /admin/
-For route creation validation tests, use a train that does not already have a route!!

Recommended test order:
1. Test demo data endpoints
2. Test route search
3. Test bookings
4. Test admin train operations
5. Test admin route operations
6. Test delay notifications
7. Test validation/error cases
